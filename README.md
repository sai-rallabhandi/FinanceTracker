Student Finance Tracker 💰📊

A Java-based desktop application developed to help students manage their personal finances efficiently. The application allows users to track income and expenses, monitor balances dynamically, and generate financial reports through an interactive and user-friendly interface.

🚀 **Features**
Add and manage income & expense transactions
Dynamic balance calculation
Delete transactions functionality
Financial report generation (Total Spending & Savings)
Persistent data storage using Java Serialization
Real-time UI updates
Modular and maintainable architecture
JUnit 5 testing for core functionalities

🛠 **Technologies Used**
Java
Java Swing
Object-Oriented Programming (OOP)
JUnit 5
Java Serialization

**📌 Design Patterns Implemented**
Singleton Pattern – Global configuration management
Observer Pattern – Automatic UI updates
Strategy Pattern – Flexible report generation
Command Pattern – Encapsulation of user actions
🧪 **Testing**
JUnit 5 was used to test:
Transaction addition
Transaction deletion
Balance calculations
Multiple transaction handling

**📂 Project Structure**
The project follows a modular package structure:
model → Data-related classes
view → User Interface components
controller → Application logic
command, strategy, observer, config → Design pattern implementations
